A little script for learning bass
By default it will pick a random note, for you to play a scale in.

Can also be used with these arguments from the command line:
-Accidentals (aliases "#" or "A") - includes sharp notes in the notes that can be chosen
-Fret (alias "F") choose a random fret instead of a note ie A 10 for the 10th fret on the A string
-Challenge (alias "C") add a random challenge like you have to start below the 12th fret
